What can I do using this website ?

This website can be used to utilise machine learning (without writing a single line of code !) to get the maximum out of your data. You just need to get your data as mentioned ahead and then you can get the most out of it by gettting predictions as well as getting to know what affects your output ! You can even use it for Kaggle competitions :)



What kind of data do I need ?

To use this website you need the following data (in csv or excel format):
1. Training Data : The set of data for which you already know the target variable (for example - Sale Price). It must contain all the data (for example : temperature affects the sale of icecream. So, if your target variable is icecream sale include temperature as a data point ) which you believe can affect the target variable and more !
2. Prediction Data : The set of data for which you want the predictions i.e for example if you know today's temperature and want to know today's expected icecream sales. To get an idea about the format in which the data should be provided check out the example given ahead.


You need to make sure that same columns are used in both training data and prediction data i.e if training data contains temperature then so must prediction data. The only difference between the data points between prediction and training must be that prediction should not contain the target variable column whereas the training should.

Don't worry if you have not understood how to use it yet. Check out the example given ahead.
